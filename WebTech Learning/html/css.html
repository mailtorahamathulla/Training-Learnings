CSS - Cascade Style Sheet


Way to interlink CSS
  1. Inline Style (Style as an attribute)
  2. Internal Style (Style as a tag)
  3. External Style (Creating a file with the extension of .css and
    import it to the html document via link tag)

Selectors are used to select or target the html element to style

Types of Selectors :
  1. Simple Selectors
    • Id name selector
    • Class name selector
    • Tag Name Selectors
    • Grouping Selector
    • Universal Selector
  2. Combinator Selectors
    • Descendant Selectors (space) - direct and indirect Sibiling
    • Child Selectors > - direct children
    • Adjacent Selectors + - next sibling
    • General Sibiling Selector ~ General sibling
  3. Attribute Selectors
  4. Pseudo Class Selector
  5. Pseudo element Selectors
